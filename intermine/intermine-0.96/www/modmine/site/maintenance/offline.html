Introduction

modMine is an integrated database of genomic, expression and protein data for Drosophila, Anopheles and C. elegans. Integrating data makes it possible to run sophisticated data mining queries that span domains of biological knowledge.

This is release 8.0 of modMine. See the release notes to find out whats new.
